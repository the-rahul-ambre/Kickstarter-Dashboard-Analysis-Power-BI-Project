# Kickstarter-Dashboard-Analysis-Power-BI-Project

## 📊 Project Overview

The **Kickstarter Dashboard** is an interactive **Power BI data visualization project** designed to analyze crowdfunding campaign performance on Kickstarter. This dashboard provides insights into project success rates, funding amounts, backer engagement, and trends across different categories, locations, and time periods.

It helps stakeholders understand key factors influencing campaign success and supports data-driven decision-making.

---

## 🎯 Project Objective

The main objectives of this dashboard are to:

* Analyze overall Kickstarter campaign performance
* Track total funding and backer contributions
* Identify the most successful project categories
* Compare success rates across goal ranges and countries
* Analyze project trends over time

---

## 📌 Key Performance Indicators (KPIs)

The dashboard highlights the following key metrics:

* **Total Backers (Successful Projects):** 40 Million
* **Total Amount Raised:** $4.3 Billion
* **Average Project Duration:** 31.6 Days

These KPIs provide a high-level overview of platform performance.

---

## 📈 Dashboard Features

### 📅 Time Analysis

**Total Projects by Year**

* Shows growth trend from 2009 to 2019
* Peak observed around 2014–2015
* Decline observed after 2018

**Success Rate Trend**

* Success rate ranges between 30% and 45%
* Shows consistency with slight fluctuations

---

### 🎯 Success Analysis by Goal Range

* Compares success percentage across funding goal ranges
* Projects with **lower funding goals (0–10K)** show higher success rates
* Higher goal projects have lower probability of success

---

### 🗂️ Category Analysis

**Total Projects by Category**

* Identifies most popular categories

**Success Rate by Category**

* Shows which categories have higher success percentage
* Helps identify profitable segments

---

### 🌍 Location Analysis

**Total Projects by Country**

Key Insights:

* United States has the highest number of projects
* Followed by UK, Canada, Australia, and Germany

---

### 📊 Project Outcome Analysis

Shows distribution of projects based on status:

* Successful
* Failed
* Cancelled
* Suspended
* Live
* Purged

Key Insight:

* Failed projects are higher than successful projects

---

## 🎛️ Filters Available

Interactive slicers allow users to filter data by:

* Year
* Quarter
* Month

This enables detailed and dynamic analysis.

---

## 🛠️ Tools & Technologies Used

* Power BI – Data Visualization
* Power Query – Data Cleaning and Transformation
* DAX – Calculated Measures and KPIs
* Excel / CSV – Dataset

---

## 💡 Key Insights

* Only a portion of projects achieve success
* Lower goal projects perform better
* US dominates crowdfunding activity
* Kickstarter experienced rapid growth between 2012 and 2015
* Category selection significantly impacts success rate
